How to Download Documents from Tarren's Website

Downloading the Portfolio

  1. Go to tarrenjohnson.info/portfolio/ and enter the code to access the portfolio
  2. Click on "File" in the top left corner, select "Print", and change the orientation to landscape if needed
  3. Check "Print backgrounds" and select "Save as PDF"
  4. Save the PDF to your computer, open it in Preview, and delete any unwanted pages

Downloading the CVs

To download the CVs, go to the links below and save the PDFs to your computer following the same steps as for the portfolio:

  • Tarren's CV: tarrenjohnson.info/cv/
  • Joel's CV: tarrenjohnson.info/cv-joel/
  • Both CVs as one document: tarrenjohnson.info/cv-both/
